# RAP - Ruby Auto Parts

## Quick start (local)

Open two terminals and run

### Backend
1. cd server
2. npm install
3. npm run dev

### Frontend
1. cd client
2. npm install
3. create .env and place this env VITE_API_BASE_URL=http://localhost:5001/api
4. npm run dev

Login using seeded accounts in server/.env (ADMIN1_EMAIL, ADMIN1_PASS, etc).
